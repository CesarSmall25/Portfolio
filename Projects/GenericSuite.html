Generic Suite

A set of plugins I've started to develop in October 2025 to help speed up and simplify game prototyping and creation

Why I made the suite

I created the suite for 3 reasons:

  • Streamline and simplify the proto steps in making games
  • Expand my portfolio
  • Sell them as easy to use, consumer-friendly plugins that work both in c++ and blueprint

What each plugin features

Out of the box experience

Each plugin is meant to be used with minimal, simple setup. Simply follow the instructions in the README and be up and running in 15 minutes.

Multiplayer support

All of the plugins are designed to have multiplayer support out of the box. For features that require a NetConnection a "networking component" is supplied for ease of use and no extra work is required. Multiplayer is also not required to use them.

Optimized

All of the plugins are designed with optimal usage in mind. While they're all still in development, plugins such as storage utilize systems like Unreal's FastArraySerializer or the Asset Manager to ensure necessary processes only and optimize the plugins efficiency.

Expandable

All of the plugins components are designed to be adjustable to the users need, from virtual or native BP functions in the components to interfaces that allow you to set custom logic.

Generic Storage

Plugin that supports a player inventory or storage containers with item stacking, custom storage sizes and a hotbar expansion system. Built with multiplayer and split screen support out of the box. Utilizes both FastArraySerializer and the Asset Manager for optimal performance.

Generic Economy

Plugin that supports wallets, shops and currencies for a game. Caches data at the start of the game to then be able to adjust and modify the values of items in the world or per shop. Utilizes the Asset Manager for optimal performance.
