Generic Crafting

Simple crafting system with no hassle in Unreal Engine

What is it?

It is a simple crafting and processing system that expands GenericStorage and allows you to define recipes and add these items to a station of any kind and construct items from others either with or without a crafting time.

How to use it

To use the system, simply create a recipe DataAsset. With that Data Asset you can define the input and output items of the recipe as well as the crafting time and the station where it can be crafted.
The crafting system itself is contained in an actorcomponent and contains a storage component from GenericStorage inside as well. With this you can set it all up according to the properties. The component has overridable functions for setting custom behaviour for crafting start, finish, and cancel.
