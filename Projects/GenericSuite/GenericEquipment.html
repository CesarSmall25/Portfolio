Generic Equipment

Easy and modular equipments or tools

What is it?

Generic Equipment is a plugin for Unreal Engine that provides a simple and flexible way to create and manage equipment or tools for characters in a game. It simply allows for a simple way to activate or attach an item to an actor and define its behavior through events and blueprints.

This system allows us to:

  • Create equipment that is visible on the player at socket points
  • Create tools with custom behaviour
  • Have replicated equipment and tools in a minimal setup

    • How to use it

    Once the plugin is installed, the only actual requirement is to add the actor component to the Character class and call TryToEquip passing in a static mesh for the visual, UObject* that implements the equipment interface for the items logic and the socket name to attach it to.

    From there the rest of the setup is up to the user in their project. They can define the equipment behaviour through the interface events in blueprint or C++ and simply need to call UseItem on the component to trigger the equipment's use behaviour (which includes a check to see if the equipment can be used at that time and delegates for feedback).
