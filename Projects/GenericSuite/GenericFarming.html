Generic Farming

Farming Framework for First and Third Person Farming Games

What is it?

Generic Farming is a plugin that provides a framework for creating farming mechanics in first and third person farming games. The reason I made this one is really more because I wanted to make samples for game types to be able to quickly toss together prototypes and show that the plugins can work together or separately. It's ultimately a way for me to be able to bundle different mechanics together for both selling as a package or just to have as samples for future projects.

It contains a few different plugins in itself:

  • Generic Economy
  • Generic Storage
  • Generic Interactions
  • Generic Building
  • Generic UI
  • Generic Characters
  • Generic Equipment
In addition to these, it also implements a few custom systems specifically for farming games such as:
  • Crop planting, growing and harvesting system
  • Soil management system

How to use it

NOTE: this system is still a work in progress and is not yet complete.

To be able to spin up a farming game quickly the user can either use the provided farming game mode and level or implement the systems from their respective plugins. For crops the user simply needs to create a new Crop Data asset defining the crop's properties such as growth time, required soil type and harvest yield.
