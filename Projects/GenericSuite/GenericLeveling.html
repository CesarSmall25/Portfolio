Generic Leveling

A quick solution to inventories and hotbars

What is it?

Generic Storage in a plugin that allows for a quick and easy way to give a player an inventory or hotbar system in their game. It is designed to be as flexible as possible so that it can be used in a variety of different game types and styles.

Generic Storage is both highly customizable and easy to use. It comes with a variety of different features that allow for a wide range of functionality, including:

  • Customizable inventory size and layout
  • Item stacking and unstacking
  • Drag and drop support through events and demos
  • Hotbar support
  • Replication using FastArraySerializer
  • Asset Manager support
