Generic Networking

Simple way to handle custom online networking for Unreal Engine

What is it?

Generic Networking provides Blueprint Async nodes that allow the user to set up quick and user friendly custom online networking solutions.

Out of the box it provides nodes for:

  • Creating Sessions
  • Hosting Sessions
  • Finding Sessions
  • Joining Sessions
  • Destroying Sessions
It also comes with a subsystem for easy access to information about data found from the nodes.

How to use it

NOTE: this system is still a work in progress and is not yet complete.

To use the plugin the user needs to set up their project for online multiplayer through the project settings and enable the Online Subsystem they wish to use (such as Steam).
Calling the nodes is as simple as dragging them into a blueprint graph and connecting them up to the desired flow. Each node has its own set of inputs and outputs that can be used to customize the experience.

For Finding sessions, the node outputs an array of session results that can be looped through to get information about each session found.
These results can then be passed into the Join Session node to connect to the desired session.
