Personal Project 2026

My first big personal project.

What is it?

While I'm deciding to keep the information on what the project is under wraps for now, I can say that it is a singleplayer, local and online co-op multiplayer game that I am developing by myself using the plugins that I've created in my Generic Suite.

When do you plan on releasing it?

My goal is to release the project before the end of this year. It's still unclear if I plan to market it or not yet but I do want to release it on Steam, which I've already started setting up and have working with their online subsystem right now.

What I learned from working on the project

So far, this project has taught me a lot about game development as a whole. From project management, to marketing, to actually developing the game itself, I've had to learn a lot of new skills and improve on existing ones. It's been a challenging but rewarding experience, and I'm excited to see where this project takes me. It has taught me so much from modularity with my plugins to scalability and performance optimizations. I've learned about things like UI, multiplayer networking, and game design principles. Overall, this project has been a great learning experience and has helped me grow as a game developer.

If ever you would like more information on this or any of my other projects, please feel free to contact me by using the contact form on the Contact Me page.
